# 📊 Sentiment Analysis Web App

A modern, high-performance web application designed to analyze the sentiment of text in real-time. Built with a focus on speed, scalability, and a premium user experience.

![Project Preview](https://raw.githubusercontent.com/mohithreddy123-hub/Sentiment_Analysis/main/preview-placeholder.png)

## 🚀 Tech Stack

- **Frontend:** React + Vite (Fast, Modern UI)
- **Styling:** Tailwind CSS (Responsive & Sleek)
- **Backend:** Supabase (Real-time Database & Auth)
- **Edge Functions:** Supabase Functions (Serverless Sentiment Logic)
- **Language:** TypeScript (Type-safe & Robust)

## ✨ Key Features

- **Real-time Analysis:** Instantly detect positive, negative, or neutral tones in text.
- **Modern UI:** A clean, minimal interface with smooth transitions.
- **Database Integration:** Store and track analysis history securely via Supabase.
- **Scalable Backend:** Powered by serverless edge functions for low-latency processing.

## 🛠️ Getting Started

### Prerequisites
- Node.js (v18 or higher)
- A Supabase account

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/mohithreddy123-hub/Sentiment_Analysis.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables:
   Create a `.env` file and add your Supabase credentials:
   ```env
   VITE_SUPABASE_URL=your_url
   VITE_SUPABASE_ANON_KEY=your_key
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## 📂 Project Structure
- `/src`: Frontend React components and logic.
- `/supabase`: Database migrations and Edge Functions.
- `/public`: Static assets.
